# 数学

## 1. 最大公约数

最大公约数 GCD（欧几里得辗转相除法）  
原理：gcd(a,b) = gcd(b, a % b)，直到余数为 0，此时 b 是最大公约数

```js
function gcd(a, b) {
    if (b === 0) return a;
    return gcd(b, a % b);
}
```

```js
function gcd(a, b) {
    while (b !== 0) {
        let temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}
```

## 3. 大数相乘

### 方法一：做加法

**核心思路：模拟手算乘法。** 把 `num2` 的每一位分别乘以 `num1` 的每一位，得到若干"部分积"，再通过补零实现错位，最后用大数加法累加起来。

以 `"123" × "45"` 为例，手算过程：

```text
    123
×    45
--------
    615   ← 123 × 5（个位，不补零）
   492    ← 123 × 4（十位，补 1 个零 → 实际是 4920）
--------
   5535   ← 615 + 4920 = 5535
```

代码拆解：

1. **外层循环** — 从右往左取 `num2` 的每一位 `y`（如 `"45"` 先取 `5`，再取 `4`）。
2. **补零（错位）** — 当 `y` 是十位时补 1 个 `0`，百位时补 2 个 `0`，依此类推。用 `curr.unshift(0)` 实现。
3. **内层循环** — `y × num1` 的每一位，`add` 变量存进位，`curr.unshift(product % 10)` 存当前位结果。
4. **累加** — 每次得到的部分积 `curr`，通过 `addStrings` 大数加法累加到最终结果 `ans` 上。

`addStrings` 辅助函数：两个指针从右往左逐位相加 + 进位，模拟手算加法。

```javascript
/**
 * @param {string} num1
 * @param {string} num2
 * @return {string}
 */
var multiply = function(num1, num2) {
    if (num1 === '0' || num2 === '0') return '0';
    let ans = '0';
    const len1 = num1.length;
    const len2 = num2.length;
    for (let i = len2 - 1; i >= 0; i--) {
        const curr = [];
        let add = 0;
        // 补后面的0
        for (let j = len2 - 1; j > i; j--) {
            curr.unshift(0);
        }
        const y = Number(num2[i]);
        for (let j = len1 - 1; j >= 0; j--) {
            const x = Number(num1[j]);
            const product = x * y + add;
            curr.unshift(product % 10);
            add = Math.floor(product / 10);
        }
        // add 最大为 8（9×9+8=89 → Math.floor(89/10)=8），永不超 9，无需取余
        if (add !== 0) {
            curr.unshift(add);
        }
        ans = addStrings(ans, curr.join(''));
    }
    return ans;
};

function addStrings(num1, num2) {
    let res = '';
    const len1 = num1.length;
    let i = len1 - 1;
    const len2 = num2.length;
    let j = len2 - 1;
    let add = 0;
    while (i >= 0 || j >= 0 || add !== 0) {
        const x = i >= 0 ? num1[i] - '0' : 0;
        const y = j >= 0 ? num2[j] - '0' : 0;
        const num = x + y + add;
        const mod = num % 10;
        add = num >= 10 ? 1 : 0;
        res = String(mod) + res;
        i--;
        j--;
    }
    return res;
}
```

### 方法二：做乘法

**核心思路：直接用数组累加，省去反复的大数加法。** 方法一每次部分积都要调用一次 `addStrings`，方法二利用 `num1[i] × num2[j]` 必然落在结果数组 `i+j` 和 `i+j+1` 两个位置的规律，一次双层循环直接得到最终结果。

以 `"123" × "45"` 为例，结果最多 `3 + 2 = 5` 位：

```text
num1:  1  2  3      (i=0,1,2)
num2:     4  5      (j=0,1)
res:  [0, 0, 0, 0, 0]   (5 位，索引 0~4)
```

每次乘积累加到 `res[i+j+1]`：

| i | j | num1[i]×num2[j] | 落到 | res 变化 |
| --- | --- | ------ | ------ | ------ |
| 2 | 1 | 3×5=15 | pos 3,4 | `[0,0,0,1,5]` |
| 1 | 1 | 2×5=10 | pos 2,3 | `[0,0,1,1,5]` |
| 0 | 1 | 1×5=5 | pos 1,2 | `[0,0,6,1,5]` |
| 2 | 0 | 3×4=12 | pos 2,3 | `[0,0,7,3,5]` |
| 1 | 0 | 2×4=8 | pos 1,2 | `[0,0,15,3,5]` → 进位 `[0,1,5,3,5]` |
| 0 | 0 | 1×4=4 | pos 0,1 | `[0,5,5,3,5]` |

去掉前导零 → `"5535"`。

```javascript
/**
 * @param {string} num1
 * @param {string} num2
 * @return {string}
 */
var multiply = function(num1, num2) {
    if (num1 === '0' || num2 === '0') return '0';
    const m = num1.length;
    const n = num2.length;
    const res = new Array(m + n).fill(0);  // 结果最多 m+n 位

    for (let i = m - 1; i >= 0; i--) {
        const x = Number(num1[i]);
        for (let j = n - 1; j >= 0; j--) {
            const y = Number(num2[j]);
            const sum = res[i + j + 1] + x * y;  // 加上原来该位置的值
            res[i + j + 1] = sum % 10;            // 低位（个位）
            res[i + j] += Math.floor(sum / 10);   // 进位加到高位
        }
    }

    // 去掉前导零
    let start = 0;
    while (start < res.length - 1 && res[start] === 0) {
        start++;
    }
    return res.slice(start).join('');
};
```

**关键点：**

- `num1[i]` 和 `num2[j]` 相乘，个位落在 `res[i+j+1]`，进位落在 `res[i+j]`。
- 直接用数组一次累加，无需像方法一那样反复 `addStrings`。
- 时间复杂度 O(m×n)，相比方法一省去了大数加法的开销。
